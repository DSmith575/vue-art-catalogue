# ArtSite - Otago Polytechnic

Art form upload

# Prerequisites

* *NodeJS v10+, npm

# Client

1. `npm i`
2. `npm run build`
3. Serve dist/

# Server

1. Spin up a MySQL/MariaDB instance
2. `create database artsite default charset utf8mb4;`
3. `cp example.env .env` and edit it
4. `npm i`
5. `npm start`
